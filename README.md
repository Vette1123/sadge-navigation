# Cool Sidebar Navigation

A slide-in side navigation menu with smooth open/close transitions. Pure HTML, CSS and JavaScript — no framework.

[**Live demo →**](https://vette1123.github.io/sadge-navigation/)

## Tech

- HTML
- CSS (transitions + transforms)
- JavaScript (no dependencies)

## Run it locally

```bash
git clone https://github.com/Vette1123/sadge-navigation.git
cd sadge-navigation
# open index.html
```
